ファイル名一括変更ツール

画像ファイルを安全に一括でリネームする Python ツールです。

対象

画像のみ

入力

先頭名

確認

y / n

安全性

上書き防止

このツールでできること

  • 画像ファイルだけを対象にして一括変更
  • 好きな先頭名を入力して連番でリネーム
  • 変更前に予定を確認
  • 同じ名前のファイルがある場合は中止

変更例

変更前

a.jpg / b.png / c.jpg

変更後

photo_001.jpg / photo_002.png / photo_003.jpg

工夫したポイント

  • .DS_Store などの不要ファイルを除外
  • 拡張子を小文字化して判定
  • 空入力なら処理を中止
  • 実行前に確認メッセージを表示
  • 同名ファイルがあれば上書きせず中止

使用技術
